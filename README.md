HackLab Application is an intentionally vulnerable web application developed for educational and laboratory purposes to demonstrate common web security vulnerabilities and their exploitation.

Implemented Vulnerabilities:
SQL Injection
Broken Access Control (BAC)
Insecure Direct Object Reference (IDOR)
Cross-Site Scripting (XSS)
Cross-Site Request Forgery (CSRF)
Local File Inclusion (LFI)
Remote Code Execution (RCE)
Brute Force Authentication

Technologies:
Node.js
Express.js
MySQL
HTML/CSS
JavaScript
